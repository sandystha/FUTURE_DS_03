##  Project Overview
College events such as technical fests, workshops, seminars, and cultural programs generate valuable student feedback. However, this feedback is often underutilized.

This project analyzes **50,000 student feedback responses** using **data analytics and NLP techniques** to extract actionable insights that can help improve future campus events.

The analysis combines **rating-based evaluation** and **text-based sentiment analysis** to understand student satisfaction and identify areas for improvement.

## Key Insights
- Technical events received the highest average ratings, followed closely by workshops and seminars.
- Cultural events showed slightly lower satisfaction, indicating execution-related challenges.
- IT and ECE departments consistently hosted higher-rated events.
- Approximately **60% of feedback is positive**, while **24% is negative**, highlighting scope for improvement.
- Career-oriented events such as **Startup Meetups** and **Cyber Security Talks** achieved the highest satisfaction scores.

---

##  Recommendations
- Improve planning and execution quality, especially for cultural and large-scale events.
- Replicate best practices from high-performing departments like IT and ECE.
- Redesign intensive formats such as hackathons and AI workshops with better structure and guidance.
- Focus more on industry-aligned and career-oriented events.
- Use sentiment analytics regularly to make data-driven improvements.
